OMII

OMI 是一个简单易学的框架，也是腾讯支持的框架，算作国产吧，需要支持！！！

OMI的学习可以进入 OMI Play

OMII = OMI + I ,多一份“I”， 多一点帮助。

现在的区别不大，OMI中已经合并OMII对于函数节点和函数表达式的扩展。

OMII完全兼容OMI，使用OMII只需继承Component类。 

      // OMII Component
        const {h,Component} =omii 
        class AComponent extends Component{

        }

异步事件

主要区别在于生命周期中的方法:beforeInstall、install、afterInstall、beforeRender、render、rendered、installed、uninstall 支持await,这源于实际工作需要，以及个人习惯！

异步、函数的使用增加了一下框架的灵活性。 

     // async install
        {
            async install(){
               this.data = await fetch(...)
            }
        }
   
     // async fetch css  
    {
       static css=()=>{
            return fetch(...css).then(r=>r.text())
       }
    }

样式扩展

OMII 增加新的样式导入属性，提供了全局主题、局部主题样式扩展。

局部样式

     // inject css to component's children  
        render(<oi-part-1 theme-css={ ... }></oi-part-1>,"body")

全局样式

     // inject css to component's children  
    const {setTheme} =omii 
    setTheme(...)
